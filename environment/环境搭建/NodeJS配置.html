1.1. 一、多版本安装

下载 nvm

选择解压版：nvm-noinstall.zip

卸载已存在的 node.js

配置环境变量

Settings 文件配置

root: D:\Environment\nvmAndNode\nvm 
path: D:\Environment\nvmAndNode\node 
arch: 64 
proxy: none
node_mirror: https://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/

NVM 使用

# 安装 node
#nvm install <version>
nvm install 12.18.2
nvm user 12.18.2
nvm ls
